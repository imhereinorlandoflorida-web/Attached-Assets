# Attached-Assets
Repository for https://replit.com/@TheArchitect128/Attached-Assets

## Setup
1. Install dependencies: `npm install`
2. Start the frontend: `npm run dev` (runs on port 8080)
3. Start the backend services: `npm start` (runs on ports 8081, 8082, 18756, 18757, 20389)

## Ports
- 8080: Vite development server (frontend)
- 8081: Express server
- 8082: Express server
- 18756: Express server
- 18757: Express server
- 20389: Express server
